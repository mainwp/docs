## GET /updates/

Returns available updates across all child sites.

**Method:** GET (readonly)

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://your-dashboard.com/wp-json/mainwp/v2/updates/?type=plugins"
```

### Parameters

| Name | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `search` | string | No | — | Search in update names |
| `include` | string | No | — | Only include these site IDs |
| `exclude` | string | No | — | Exclude these site IDs |
| `type` | string | No | all | Filter: wp, plugins, themes, translations |

### Response

```json
{
  "total_updates": 15,
  "sites": [
    {
      "id": 1,
      "url": "https://example.com",
      "wp_update": true,
      "plugin_updates": 3,
      "theme_updates": 1,
      "translation_updates": 2
    }
  ]
}
```

## GET /updates/{site}

Returns available updates for a specific child site.

**Method:** GET (readonly)

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://your-dashboard.com/wp-json/mainwp/v2/updates/1"
```

### Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `site` | integer/string | Yes | Site ID or domain |
| `search` | string | No | Search in update names |
| `type` | string | No | Filter: wp, plugins, themes, translations |

### Response

```json
{
  "site_id": 1,
  "site_url": "https://example.com",
  "updates": {
    "wp": {
      "current": "6.4.2",
      "new": "6.4.3"
    },
    "plugins": [
      {
        "slug": "akismet/akismet.php",
        "name": "Akismet",
        "current_version": "5.3",
        "new_version": "5.3.1"
      }
    ],
    "themes": [],
    "translations": []
  }
}
```

## POST /updates/update/

Triggers updates for all sites. Responds immediately; updates run in the background.

**Method:** POST

<Warning>
This action applies updates across all sites. Ensure you have recent backups before proceeding.
</Warning>

```bash
curl -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  "https://your-dashboard.com/wp-json/mainwp/v2/updates/update/?type=plugins"
```

### Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `type` | string | No | Limit to: wp, plugins, themes, translations |

### Response

```json
{
  "success": 1,
  "message": "Batch updates all started successfully.",
  "last_time_start_batch_updates": "2024-01-15T10:30:00"
}
```

## POST /updates/{site}/update/plugins

Updates plugins on a single site.

**Method:** POST

```bash
# Update all plugins
curl -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  "https://your-dashboard.com/wp-json/mainwp/v2/updates/1/update/plugins"

# Update specific plugin
curl -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  "https://your-dashboard.com/wp-json/mainwp/v2/updates/1/update/plugins?slug=akismet/akismet.php"
```

### Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `site` | integer/string | Yes | Site ID or domain |
| `slug` | string | No | Specific plugin slug to update; omit for all |

### Response

```json
{
  "success": true,
  "site_id": 1,
  "updated": ["akismet/akismet.php", "jetpack/jetpack.php"],
  "failed": [],
  "message": "Plugins updated successfully"
}
```

## POST /updates/ignore

Ignores updates for specified plugins or themes.

**Method:** POST

```bash
curl -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"type": "plugin", "slug": "custom-plugin/custom-plugin.php"}' \
  "https://your-dashboard.com/wp-json/mainwp/v2/updates/ignore"
```

### Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `type` | string | Yes | plugin or theme |
| `slug` | string | Yes | Plugin or theme slug |
| `site_id` | integer | No | Ignore only for specific site |

### Response

```json
{
  "success": true,
  "message": "Updates ignored for custom-plugin/custom-plugin.php"
}
```
