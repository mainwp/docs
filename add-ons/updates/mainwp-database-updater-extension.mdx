The Database Updater Extension detects if there are updates to WooCommerce and Elementor plugin databases. This means that you no longer need to access the wp-admin of a Child Site just to click the Update Database button for WooCommerce and Elementor.

## Install the Database Updater Extension

<Steps>
<Step title="Go to your MainWP Dashboard">
</Step>
<Step title="Go to the MainWP > Extensions page">
</Step>
<Step title="Click the Install Extensions button">
Click the **Install Extensions** button [![MainWP Domain Monitor Extension 1](/images/add-ons/2833252556cc.png)
</Step>
<Step title="Locate the Database Updater extension and select it">
</Step>
<Step title="Click the Install button">
</Step>
<Step title="Once the installation process is complete, click the Activate Extensions button.">
Once the installation process is complete, click the **Activate Extensions** button. ![MainWP Domain Monitor Extension 2](/images/add-ons/578da5c7dc89.png)
</Step>
</Steps>


## How to Update the Databases

After you've installed the extension, a new Database Updates page will appear. ![Database Updates page showing WooCommerce and Elementor update status](/images/add-ons/database-updater-settings.png) The page can be accessed by going to the **Dashboard > Updates > Database Updates** or **Dashboard > Extensions > Database Updater** menu entry.

### Update databases on all child sites

<Steps>
<Step title="Go to the MainWP > Updates > Database Updates page.">
</Step>
<Step title="Click the Update All button in the top right">
![Click the Update All button in the top right](/images/add-ons/295e9ab3a808.png)

</Step>
</Steps>


### Update databases on individual child sites

<Steps>
<Step title="Go to the MainWP > Updates > Database Updates page.">
</Step>
<Step title="Expand the plugin row and locate the desired site in the table">
</Step>
<Step title="Click the Update Now button next to the desired site">
![Click the Update Now button next to the desired site](/images/add-ons/database-updater-overview.png)

</Step>
</Steps>
