The WPvivid Backup integration for MainWP allows you to manage backups across your child sites using the popular WPvivid Backup plugin, directly from your MainWP Dashboard.

## About WPvivid Backup

[WPvivid Backup](https://wpvivid.com/) is a free WordPress backup plugin offering scheduled backups, cloud storage integration, and easy site migration capabilities.

## Key Features

- Create and manage backups across multiple sites
- Schedule automated backups
- Store backups in cloud storage (Google Drive, Dropbox, Amazon S3, etc.)
- Easy site migration and staging
- View backup status from MainWP Dashboard

## Requirements

- WPvivid Backup plugin installed on child sites
- MainWP Dashboard with Child plugin connected

## Installation

<Steps>
<Step title="Install WPvivid on child sites">
Install and activate the [WPvivid Backup plugin](https://wordpress.org/plugins/wpvivid-backuprestore/) on your child sites.
</Step>
<Step title="Configure backup settings">
Set up your backup schedules and cloud storage destinations in each child site.
</Step>
<Step title="Sync your MainWP Dashboard">
Sync your MainWP Dashboard to see WPvivid status across your network.
</Step>
</Steps>
