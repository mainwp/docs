<Info>
**Integration Add-on** - This add-on extends MainWP to work with Atarim. Requires Atarim to be installed on your child sites.
</Info>

Atarim is a service for collaborating with your clients. Two-way integration offers an insight into website status on Atarim Dashboard and an insight into Atarim tasks on the MainWP Dashboard. Please note that this integration requires a [paid Atarim plan](https://atarim.io/pricing/) (Starter, Expert, or Agency).

## Connect Atarim to MainWP Dashboard

<Steps>
<Step title="Install Atarim Extension">
[Install](/dashboard/overview/manage-extensions#install-extensions) Atarim [Extension](https://mainwp.com/extension/atarim/)
</Step>
<Step title="Go to Atarim Dashboard > Settings > Integrations Settings page">
</Step>
<Step title="Enable MainWP Integration and copy the API Key">
Enable MainWP Integration and copy the API Key ![Atarim & MainWP integration API Key](/images/add-ons/67bcddbb05b0.png)
</Step>
<Step title="Go to MainWP Dashboard > Extensions > Atarim > Settings page">
</Step>
<Step title="Paste the API Key">
</Step>
<Step title="Click the Save Settings button">
Click the Save Settings button ![Connect MainWP to Atarim settings](/images/add-ons/42178f40ecfd.png)
</Step>
</Steps>


After connecting with Atarim, MainWP will gain the following features:

-   **Manage Sites table**
    -   A new column will be visible showing the number of Atarim tasks for child sites. Clicking the icon will reveal additional information about those tasks. ![Atarim Tasks modal element](/images/add-ons/782c77e43684.gif)
-   **Overview widgets**
    -   On the Overview and Individual Child site overview, a new widget will display information about Atarim tasks ![Atarim tasks widget](/images/add-ons/atarim-tasks-widget.png)
-   **Pro Reports data Note**: Pre Reports extension version 4.0.11 is required for Atarim data
    -   Atarim Data can be included in the Report from the **Report Data** section ![Atarim pro reports settings](/images/add-ons/3e3c796fdd7f.png)
    -   Atarim data can also be included by using two new tokens \[atarim.all.tasks\] - Displays all tasks for a site within the selected date range \[atarim.billable.tasks\] - Displays all billable tasks for a site within the selected date range

<Steps>
<Step title="Go to MainWP Dashboard > Sites > Manage Sites page">
</Step>
<Step title="Click the Cog icon (Page Settings) in the top right">
</Step>
<Step title="De-select Atarim in the Show Columns section">
De-select Atarim in the **Show Columns** section ![Remove Atarim column from the Manage Sites table.](/images/add-ons/atarim-hide-column.gif)
</Step>
<Step title="Click the Save Settings button">
</Step>
</Steps>


### How to unset the Atarim column

Insert the following snippet using our [Custom Dashboard](https://mainwp.com/extension/mainwp-custom-dashboard-extension/) extension, or by inserting it in the `functions.php` file of the active theme of your MainWP Dashboard:

```php
add_filter( 'mainwp_sitestable_getcolumns', 'myhook2_mainwp_sitestable_getcolumns', 20, 2 );
function myhook2_mainwp_sitestable_getcolumns( $columns ) {
	if(is_array($columns) && isset($columns\['atarim_tasks'\])){
		unset( $columns\['atarim_tasks'\] );
	}
	return $columns;
}

```
