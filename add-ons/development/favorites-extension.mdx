The **MainWP Favorites Extension** allows you to save frequently used plugins and themes for quick and easy installation on your child sites. Instead of manually searching for and uploading plugins or themes each time, you can store your preferred items in your Favorites list and install them whenever needed.

## Accessing the Favorites Extension

![Favorites Extension page showing saved plugins and themes lists](/images/add-ons/favorites-plugins-themes.png)

To access the Favorites Extension, navigate to:

**Dashboard > Add-ons > Favorites**

Here, you will see two sections:

-   **Plugins** – A list of your favorite plugins
-   **Themes** – A list of your favorite themes

## Adding Plugins and Themes to Favorites

![Animation showing how to add a new plugin or theme to Favorites](/images/add-ons/favorites-add-new.gif)

To add a plugin or theme to your favorites list:

<Steps>
<Step title="Click the Add New button.">
</Step>
<Step title="Paste the URL of the ZIP file for the plugin or theme.">
Paste the URL of the ZIP file for the plugin or theme.
    -   You can use a WordPress.org URL or a custom URL where your custom plugin/theme is stored. **NOTE**: The URL must be publically accessible.
</Step>
<Step title="(Optional) Define a Name for the item.">
(Optional) Define a **Name** for the item.
    -   If left blank, the system will automatically fetch the name from the item's metadata.
</Step>
<Step title="Click Save to add it to your favorites.">
</Step>
</Steps>


The process is the same for both plugins and themes.

## Managing Your Favorites List

![Animation showing how to edit, download, and delete favorites](/images/add-ons/favorites-edit-delete.gif)

Once you've added items to your Favorites list, you can manage them using the following options:

-   **Edit**: Click the **pen icon** to update the name or URL.
-   **Download**: Click the **download icon** to save the ZIP file to your local computer.
-   **Delete**: Click the **trash icon** to remove the item from your favorites.

## Installing Favorites on Child Sites

![Animation showing how to install favorites on child sites](/images/add-ons/favorites-install.gif)

You can install your saved favorites directly on your child sites from the **Install Plugins** and **Install Themes** pages.

To install a favorite:

<Steps>
<Step title="Navigate to one of the following:">
Navigate to one of the following:
    -   **Dashboard > Plugins > Install Plugins**
    -   **Dashboard > Themes > Install Themes**
</Step>
<Step title="Click the Install Favorites button.">
</Step>
<Step title="Select the plugins or themes you want to install.">
</Step>
<Step title="Choose the child sites where you want to install the selected items.">
</Step>
<Step title="Click Complete Installation to start the process.">
</Step>
</Steps>


MainWP will then install the selected items on the chosen sites automatically.
