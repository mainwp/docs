The MainWP Early Access Extension gives you access to upcoming MainWP features and extensions before they're publicly released. This allows you to test new functionality, provide feedback, and prepare for upcoming changes.

## Key Features

- Access beta versions of new MainWP extensions
- Preview upcoming Dashboard features
- Provide early feedback to shape development
- Prepare your workflow for upcoming changes

## Installation

The MainWP Early Access Extension needs to be installed only on your MainWP Dashboard site. You can [install it as any other MainWP Extension](/dashboard/overview/manage-extensions#install-extensions).

## Getting Started

<Steps>
<Step title="Install and activate the extension">
Navigate to **MainWP > Extensions** and install the Early Access extension.
</Step>
<Step title="Configure your preferences">
Go to **MainWP > Extensions > Early Access** to configure which beta features you want to enable.
</Step>
<Step title="Test new features">
Once enabled, early access features will appear in your Dashboard for testing.
</Step>
</Steps>

<Warning>
Early access features may be unstable or incomplete. Use them in testing environments before deploying to production sites.
</Warning>
