The [MainWP Comments Extension](https://mainwp.com/add-on/comments/) allows you to Approve, Disapprove, Mark as spam, Un-spam, Edit, and Delete comments in all your child sites from one place. By using bulk actions, you will save time for more important tasks, or if you prefer, you can still moderate your comments one by one.

## The Recent Comments Widget

A great addition to this extension is the **Recent Comments** widget. You will always be able to track the latest Approved, Pending, and Trashed comments on your sites. You can also use the provided links to quickly moderate the latest comments.

## Manage Comments

The MainWP Comments Extension allows you to manage comments on your Websites. It will allow you to remotely

-   Approve
-   Disapprove
-   Mark as spam
-   Un-spam
-   Edit
-   Delete

On the **MainWP > Add-ons > Comments** page, you can easily find all existing comments and manage them without of hassle of logging into each child site separately.

<Steps>
<Step title="Log in to your MainWP Dashboard">
</Step>
<Step title="Go to the MainWP > Extensions > Comments page">
</Step>
<Step title="Select wanted Child Sites">
</Step>
<Step title="In the Search Options section, set your search preferences">
In the **Search Options** section, set your search preferences
    1.  **Status** filters will allow you to narrow down the search to only selected comment statuses
    2.  **The keyword** field will allow you to find comments where an entered word is used
    3.  **Date Range** will narrow down the search to comments that have been submitted in the selected date range
</Step>
<Step title="Click the Show Comments button">
</Step>
</Steps>


![MainWP Comments Add-on 2025](/images/add-ons/comments-manage-list.jpg) By default, the MainWP Dashboard plugin is set to return (show) a maximum of 50 comments from one Child Site.

### Delete Comment Permanently

Since WordPress allows you to permanently delete only comments that are currently in Trash, before you can permanently delete them, you need to send them to Trash. Once they are in Trash, the **Delete Permanently** action will be available.
