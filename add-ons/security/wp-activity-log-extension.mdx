The WP Activity Log integration for MainWP allows you to monitor user activity, track changes, and maintain audit logs across your child sites from your MainWP Dashboard.

## About WP Activity Log

[WP Activity Log](https://melapress.com/wordpress-activity-log/) is the most comprehensive WordPress activity log plugin, tracking changes to posts, pages, users, plugins, themes, and more.

## Key Features

- Complete activity log for all WordPress changes
- User session management
- Real-time notifications for critical changes
- Search and filter activity logs
- Export logs for compliance reporting
- Track login/logout activity
- Monitor plugin and theme changes
- Database activity logging

## Requirements

- WP Activity Log plugin installed on child sites
- MainWP Dashboard with Child plugin connected

## Installation

<Steps>
<Step title="Install WP Activity Log on child sites">
Install and activate the [WP Activity Log plugin](https://wordpress.org/plugins/wp-activity-log/) on your child sites.
</Step>
<Step title="Configure logging settings">
Set up which activities to track and configure notification preferences.
</Step>
<Step title="Sync your MainWP Dashboard">
Sync your MainWP Dashboard to see activity logs across your network.
</Step>
</Steps>
