The Security Ninja integration for MainWP allows you to monitor security status and run security audits across your child sites from your MainWP Dashboard.

## About Security Ninja

[Security Ninja](https://wordpress.org/plugins/security-ninja/) is a comprehensive WordPress security plugin that performs security tests, detects vulnerabilities, and helps harden your WordPress installations.

## Key Features

- Run 50+ security tests on your sites
- Detect vulnerabilities and misconfigurations
- Core file integrity checking
- Malware scanning
- Security hardening recommendations
- Scheduled security scans
- Email notifications for security issues

## Requirements

- Security Ninja plugin installed on child sites
- MainWP Dashboard with Child plugin connected

## Installation

<Steps>
<Step title="Install Security Ninja on child sites">
Install and activate the [Security Ninja plugin](https://wordpress.org/plugins/security-ninja/) on your child sites.
</Step>
<Step title="Run initial security scan">
Perform an initial security scan on each child site to establish a baseline.
</Step>
<Step title="Sync your MainWP Dashboard">
Sync your MainWP Dashboard to see security status across your network.
</Step>
</Steps>
