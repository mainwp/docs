The SEOPress for MainWP is a free extension that allows you to manage the SEO parameters of all connected Child Sites on which you have installed the SEOPress plugin! Please note that you need to have the SEOPress plugin on the MainWP Dashboard website and at least on one Child Site. The minimum versions supported:

-   SEOPress Free (5.9+)
-   SEOPress PRO (5.9+)
-   MainWP Dashboard (4.2.7+)
-   MainWP Child (4.2.6+)

## Install SEOPress Extension on MainWP Dashboard

The SEOPress extension is available on the WordPress repository. Installing it is just like installing a regular plugin.

1.  From inside the wp-admin of your MainWP Dashboard, navigate to **Plugins -> Add New**.
2.  Search for “**SEOPress for MainWP**”.
3.  Install and Activate the plugin.
4.  Get into your MainWP Dashboard, navigate to **Extensions -> SEOPress**, and start using it. ![SEOPress Dashboard page](/images/miscellaneous/add-ons-posts-pages-f01810.png)
