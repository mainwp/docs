This article explains how to get MainWP Essentials and activate your API key on your MainWP Dashboard, which allows you to install extensions directly and enable automatic updates.

## What You'll Learn

- How to create a MainWP account
- How to get MainWP Essentials (free)
- How to copy and activate your API key
- How to install free extensions

---

## Create or Log Into Your MainWP Account

<Steps>
<Step title="Go to MainWP.com and click the Go to my account link">
Go to [MainWP.com](https://mainwp.com) and click the **[Go to my account](https://mainwp.com/my-account/)** link at the top-right of the page.

![Arrow pointing to Go to My Account link on mainwp.com homepage.](/images/miscellaneous/advanced-miscellaneous-447eb4.png)
</Step>
<Step title="Log in or create your MainWP Account">
If you already have a MainWP Account, use the Login Form (on the left side) to log into your account. If you do not, use the Register form (on the right side) to create your MainWP Account.
</Step>
</Steps>

## Get MainWP Essentials

<Steps>
<Step title="Go to Sign Up page and click Get MainWP Essentials">
Once you are logged in, go to the [**Sign Up**](https://mainwp.com/signup/) page and click the **Get MainWP Essentials** button.

![Arrow pointing to Get MainWP Essentials button on the Sign Up page.](/images/miscellaneous/advanced-miscellaneous-1724ff.png)
</Step>
<Step title="Complete checkout">
On the **Checkout** page, verify your details and proceed.

<Note>Payment method is **NOT** needed for MainWP Essentials.</Note>
</Step>
</Steps>

## Copy Your API Key

<Steps>
<Step title="Navigate to your Account Extensions page">
Navigate to your **[Account > Extensions](https://mainwp.com/my-account/my-api-downloads/)** page. Here you will find your Main API Key.
</Step>
<Step title="Copy your Main API Key">
Copy your Main API Key - you'll need it in the next section.

![MainWP main API key in the My Account page.](/images/miscellaneous/advanced-miscellaneous-4057bc.png)
</Step>
</Steps>

## Activate Your API Key in MainWP Dashboard

<Steps>
<Step title="Navigate to your MainWP Dashboard site">
Go to your self-hosted MainWP Dashboard site.
</Step>
<Step title="Go to Dashboard > Extensions">
Navigate to the **Dashboard > Extensions** page.
</Step>
<Step title="Find the Install and Activate Extensions box">
In the top-right of the page, you will find the **Install and Activate Extensions** box.
</Step>
<Step title="Paste your API Key">
Paste your Main API Key and check **Remember MainWP Main**.
</Step>
<Step title="Validate your API Key">
Click the **Validate my MainWP Main API Key** button.

![Click the Validate my MainWP Main API Key button](/images/miscellaneous/advanced-miscellaneous-785926.gif)
</Step>
</Steps>

## Install Extensions

<Steps>
<Step title="Click Install Extensions">
Click the **Install Extensions** button.
</Step>
<Step title="Select and install your extensions">
Select the extensions you want to install and click the **Install Selected Extensions** button.

![Extension selection modal showing available free extensions](/images/miscellaneous/advanced-miscellaneous-a0b5e2.png)
</Step>
</Steps>
