MainWP Dashboard pages can be customized with different widget arrangements and table column configurations. If you want to restore a page to its default layout, you can reset the settings.

## What You'll Learn

- How to reset widget layouts on overview pages
- How to reset table layouts on list pages

---

## When to Reset Page Settings

Use this feature when:

- Widgets are arranged in an unhelpful way
- Important widgets are hidden
- Table columns are in a confusing order
- You want to start fresh with default settings

---

## Reset Widget Layout

Widget-based pages include the Overview Page, Child Site Overview Page, and Client Card page.

<Steps>
<Step title="Open page settings">
Click the **Cog** icon in the top right corner of the page.
</Step>
<Step title="Reset the page">
Click the **Reset Page** button in the bottom left of the popup modal.
</Step>
<Step title="Confirm">
Click **Yes, Proceed!** to confirm.
</Step>
</Steps>

![Reset page settings](/images/miscellaneous/advanced-miscellaneous-8d2203.gif)

This resets:
- Widget positions
- Widget sizes
- Widget visibility (shown/hidden)

---

## Reset Table Layout

Table-based pages include Manage Sites, Manage Plugins, Manage Themes, and similar list views.

<Steps>
<Step title="Open page settings">
Click the **Cog** icon in the top right corner of the page.
</Step>
<Step title="Reset the table">
Click the **Reset Page** button in the bottom left of the popup modal.
</Step>
<Step title="Confirm">
Click **Yes, Proceed!** to confirm.
</Step>
</Steps>

![Reset table layout](/images/miscellaneous/advanced-miscellaneous-5b6336.png)

This resets:
- Column visibility
- Column order
- Any sorting preferences
