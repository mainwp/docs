---
title: How to Install MainWP Dashboard App on Cloudways Hosting
description: Deploy WordPress with pre-installed MainWP Dashboard on Cloudways managed hosting using their one-click application option.
---

Cloudways offers a WordPress application with MainWP Dashboard pre-installed for quick deployment. This option is enabled by default for users who sign up through the MainWP Hosting page. Existing Cloudways users may need to contact Cloudways support to enable this option.

## What You Will Accomplish

- Deploy a new Cloudways server with WordPress and MainWP Dashboard pre-installed
- Access your MainWP Dashboard and complete the Quick Setup Wizard
- Have a working MainWP Dashboard ready to connect child sites

## Prerequisites

- A Cloudways account (new or existing)
- Payment method for Cloudways hosting
- No domain name required (Cloudways provides a temporary URL)

<Note>
Existing Cloudways users: If you don't see the **WordPress (with MainWP)** option in your Applications menu, contact Cloudways support to have it enabled.
</Note>

## Deploy MainWP Dashboard on Cloudways

<Steps>
<Step title="Go to the MainWP Hosting page">
Visit the [MainWP Hosting](https://mainwp.com/hosting/) page and click **Get Started Here**.

![MainWP Hosting page with Get Started Here button for Cloudways signup](/images/miscellaneous/dashboard-overview-6b58e8.jpg)

<Note>Sign up through this page to have the WordPress (with MainWP) option enabled by default.</Note>
</Step>
<Step title="Sign up for a Cloudways account">
Complete the Cloudways registration process.
</Step>
<Step title="Log in to your Cloudways account">
After registration, log in to your Cloudways dashboard.
</Step>
<Step title="Go to the Applications section">
Navigate to the **Applications** section.

![Cloudways - Applications](/images/miscellaneous/dashboard-overview-fa392d.jpg)
</Step>
<Step title="Click Launch">
Click the **Launch** button.

![Cloudways - Launch Application](/images/miscellaneous/dashboard-overview-a2c704.jpg)
</Step>
<Step title="Select your server provider">
On the **Application & Server Details** page, choose your preferred server provider (e.g., Vultr, DigitalOcean, AWS).

![Cloudways - Vultr](/images/miscellaneous/dashboard-overview-da7287.jpg)
</Step>
<Step title="Select WordPress (with MainWP)">
From the **Application** dropdown menu, choose **WordPress (with MainWP)**.

![Cloudways Application menu showing WordPress with MainWP option](/images/miscellaneous/dashboard-overview-fe04e9.jpg)
</Step>
<Step title="Configure and launch">
Name your Application, Server, and Project, then click **Launch Now**.

![Cloudways - Launch MainWP App](/images/miscellaneous/dashboard-overview-23211d.jpg)
</Step>
<Step title="Wait for server creation">
The server creation process takes a few minutes to complete.

![Cloudways - Creating Server](/images/miscellaneous/dashboard-overview-db65ce.jpg)
</Step>
<Step title="Access your application">
Click **Applications** (www) in the top menu.

![Cloudways - Applications menu](/images/miscellaneous/dashboard-overview-5d723a.jpg)
</Step>
<Step title="Select your MainWP Dashboard application">
Click your MainWP Dashboard application from the popup list.

![Cloudways - My MainWP Dashboard](/images/miscellaneous/dashboard-overview-c64c18.jpg)
</Step>
<Step title="Find your login credentials">
Locate the **Admin Panel** section to find your application URL, username, and password.

![Cloudways - Admin Panel](/images/miscellaneous/dashboard-overview-1cc673.jpg)
</Step>
<Step title="Log in to WordPress">
Open the Admin Panel URL and log in with the provided credentials.

![Cloudways - Dashboard Login](/images/miscellaneous/dashboard-overview-7638ae.jpg)
</Step>
<Step title="Start the Quick Setup Wizard">
After logging in, the MainWP Quick Setup Wizard appears. Click **Let's Go**.

![MainWP Quick Setup Wizard welcome screen with Let's Go button](/images/miscellaneous/dashboard-overview-d09f0b.jpg)
</Step>
<Step title="Select your installation type">
On the **Installation** step, select **Web host** and click **Continue**.

![Quick Setup Wizard Installation step with Web host option selected](/images/miscellaneous/dashboard-overview-7c51b9.jpg)
</Step>
<Step title="Verify system requirements">
Review the system requirements check to confirm all requirements are met.

![Quick Setup Wizard showing system requirements verification](/images/miscellaneous/dashboard-overview-fed1ff.jpg)
</Step>
<Step title="Complete the Quick Setup Wizard">
Continue through the remaining wizard steps. For detailed guidance, see [Get Started with MainWP](/getting-started/get-started-with-mainwp).
</Step>
<Step title="Update MainWP Dashboard plugin">
Go to **WP Admin > Plugins** and check if a MainWP Dashboard update is available. If so, update the plugin.

![Update MainWP Dashboard](/images/miscellaneous/dashboard-overview-8c696c.jpg)
</Step>
</Steps>

## Migrating from Another Host

If you have an existing MainWP Dashboard on different hosting and want to migrate to Cloudways:

1. [Export your child sites](/sites/management/manage-child-sites) from your current Dashboard
2. Set up your new MainWP Dashboard on Cloudways using the steps above
3. [Import your sites](/sites/management/manage-child-sites) to the new Dashboard
